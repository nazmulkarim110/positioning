CSS Position

The position property specifies the type of positioning method used for an element.

Position Types / values: 5 (Five)

  1. static
  2. relative
  3. fixed
  4. absolute
  5. sticky

Elements are then positioned using the top, bottom, left, and right properties. However, first need to set position property.

CSS z-index

The z-index CSS property sets the z-order of a positioned element and its descendants.

Overlapping elements with a larger z-index cover those with a smaller one.
